## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | hsl(var(--background)) | Background, main surfaces |
| Secondary (30%) | hsl(var(--card)) | Cards, panels, sidebar |
| Accent (10%) | hsl(var(--primary)) | Primary buttons, active state toggles (#6C3FF5 used in login) |
| Destructive | hsl(var(--destructive)) | Destructive actions (Cancel appointment) |
